08-05-24 Page system test

My thoughts ended up being true so I got my page system working. It works by dividing the amount of available fishes by the max number of fishes per page. Then for each page create a container of divs. The range of fishes (divs) is given by the formula:

( (pageIndex + 1) * fishesPerPage - fishesPerPage, (pageIndex + 1) * fishesPerPage )

So in my case it goes from (0 - 7), (8 - 15), (16 - 23)... Being pageIndex [0,1,2,3,4,5...] and 8 fishes per page.
